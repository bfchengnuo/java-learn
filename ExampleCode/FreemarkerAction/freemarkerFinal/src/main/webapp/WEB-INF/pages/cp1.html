取值（插值）

1，取基本值

  • 整数：${intVar}
    长整数：${longVar}
    字符串：${stringVar}
    双精度：${doubleVar}
    布尔值：${booleanVar?string('yes','no')}
    日期：${dateVar?string('yyyy-MM-dd HH:mm:ss')}
  • 不存在的值 或者 为null的值
    • ${abc!0}
  • 对象
    • ${(userObj.name)!}
    ${(userObj.brief)!}
  • 集合 List
    • <#list myList as item> ${item!}
  • 集合 map
    • <#list map?keys as key> ${key} : ${map[key]}
